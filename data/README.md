\# Data



This analysis requires two input files:



1\. `feature\_table\_5000.tsv`

&#x20;  - Amplicon feature/OTU table

&#x20;  - Samples should be represented as columns before transposition in the script



2\. `Healthy\_0-4\_agebins.txt`

&#x20;  - Metadata file

&#x20;  - Must contain a `sample-id` column

&#x20;  - Must contain an `Age\_bin4` column for age-bin classification



Raw data are not included in this repository due to data access and privacy considerations.
